## 演算子の「順番」と「優先順位」

### 📘 評価順序とは？

Java は**左から右に**評価されるのが基本ですが、**演算子の優先順位**があるため、先に計算されるものがる

### 🥇 優先順位の例

| 優先度 | 演算子  | 説明                 |
| ------ | ------- | -------------------- |
| 高     | `()`    | 括弧内を最優先で計算 |
| 中     | `* / %` | 掛け算、割り算、余り |
| 低     | `+ -`   | 足し算、引き算       |

例：

```java
int result = 3 + 4 * 2; // → 11（先に4*2が計算される）
int result = (3 + 4) * 2; // → 14（括弧が優先）
```

### 💡 ヒント

よく分からないときは「`()`（括弧）」で明確にするとバグ防止になる
