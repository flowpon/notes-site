## Javaで使う演算子とは？

演算子とは「計算や処理を行う記号」のこと

### 🔢 算術演算子（数値の計算）

| 記号 | 意味 | 例 |
|------|------|----|
| + | 足し算 | `a + b` |
| - | 引き算 | `a - b` |
| * | 掛け算 | `a * b` |
| / | 割り算 | `a / b` |
| % | 余り | `a % b` |

### 🧮 代入演算子

- `=` 右側の値を左の変数に代入します。
- 複合代入（例： `a += 3` は `a = a + 3` と同じ）

### 🤝 比較演算子（値を比べる）

| 記号 | 意味 | 例 |
|------|------|----|
| == | 等しい | `a == b` |
| != | 等しくない | `a != b` |
| > | より大きい | `a > b` |
| < | より小さい | `a < b` |
| >= | 以上 | `a >= b` |
| <= | 以下 | `a <= b` |

### 🧠 論理演算子（条件を組み合わせる）

| 記号 | 意味 | 例 |
|------|------|----|
| && | AND（かつ） | `a > 0 && b > 0` |
| \|\| | OR（または） | `a > 0 \|\| b > 0` |
| ! | NOT（反転） | `!(a > 0)` |
