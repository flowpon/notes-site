## Java の型変換とは？

Java では**異なる型を変換（キャスト）**して使うことがある

### ✨ 暗黙的キャスト（自動で変換される）

```java
int a = 10;
double b = a; // 自動でint → doubleに変換
```

小さい型 → 大きい型への変換は OK（例：`int → long`、`float → double`）

### ⚠️ 明示的キャスト（自分で変換する）

```java
double a = 3.14;
int b = (int) a; // 明示的にキャスト → 小数点以下が切り捨てられる
```

大きい型 → 小さい型は明示的に変換が必要

情報が失われる可能性あり（＝データ損失）
