# Java の変数とデータ型について

Java では、変数を使う前に**データ型**を指定して宣言します。これは、メモリの使い方や処理の正確さを保つために重要です。

## 変数の宣言と初期化

```java
int age = 25;
String name = "Hana";
```

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("こんにちは、世界");
    }
}
```
